Strings Internationalization API for ColdFusion

Welcome!

The ability for software to support multiple locales is becoming a more commonplace requirement in todays modern web applications. This feature is more commonly referred to as either Internationalization, Localization or i18n.

The Strings Internationalization API for CF (CF SI) is intended to get you quickly on your way to a highly accessible and localized RIA by providing an easy to understand and even easier to implement API.
